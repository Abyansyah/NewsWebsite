# Website AdaApA.

Website news berita Indonesia

## Run Locally

Clone the project

```bash
  git clone https://github.com/destiaww/AdaApa-nextjs.git
```

Go to the project directory

```bash
  cd AdaApa-nextjs
```

Install dependencies

```bash
  npx create-next-app .
```

Start the server

```bash
  npm run dev
```
