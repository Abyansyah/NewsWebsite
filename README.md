# Project Title

AdaApa News

## Run Locally

Clone the project

```bash
  git clone https://github.com/destiaww/AdaApa-nextjs.git
```

Go to the project directory

```bash
  cd AdaApa-nextjs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Build With

This project is used by the following companies:

- Next Js
- Node Js
- CSS Module

## Deployment

To deploy this project run

adaapa.vercel.app

- Search by title
- Short by category
